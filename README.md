# Student Result Management Service

## Requirements

- Docker
- Docker Compose

## How to install

Before installing, make sure you have your `.env` accordingly
```
cp .env-sample .env
```

```sh
make lock
```

## How to build

```sh
make build
```


## How to run

```sh
make run
```
